# Shree Shankar Traders & Painting Services

A full-stack e-commerce web application for a local paint shop.

## Features
- Product catalogue with live colour visualiser
- Cart system with Razorpay (UPI/Card) + Cash on Delivery
- Service booking system with room-wise area estimator
- PDF invoice generation per order
- Real-time delivery tracking with staff assignment
- Admin dashboard — products, orders, deliveries, bookings, staff

## Tech Stack
Python 3.13 · Django 5.2 · MySQL 8.4 · Bootstrap 5 · Razorpay

## Setup
1. Clone the repo
2. Create a `.env` file with your credentials (see Step 5 above)
3. Run `pip install -r requirements.txt`
4. Import `recreate_db.sql` in MySQL
5. Run `python manage.py runserver`
